# AI Native Product Builder Portfolio

Minimal Astro + TypeScript foundation for the AI Native Product Builder Portfolio.

This app is intentionally small:

- Astro static-first shell
- MDX content collection with Zod schema
- Tailwind CSS v4 through the Vite plugin
- Minimal Home, Projects, and Case Study routes
- Placeholder content only for schema and routing validation

Deferred:

- Full visual system implementation
- Real case study content
- CMS, login, database, AI runtime, and complex demos
