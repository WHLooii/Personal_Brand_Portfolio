## 评估标准

| 维度 | 说明 |
| --- | --- |
| 用户痛点强度 | 问题是否真实、高频、明确 |
| MVP 清晰度 | 是否能拆成小步可验证版本 |
| 技术可行性 | 当前能力和资源是否足够推进 |
| 商业可能性 | 是否存在明确付费、降本或增效价值 |
| 风险 | 市场、技术、数据、合规或交付风险 |
